# PA2-Hash-Table

Concurrent Hash Table Implementation

This program implements a concurrent hash table that allows multiple threads to perform on a shared collection of key-data pairs.

Files:
- chash.c: Main program file
- hash_table.h: Header file for hash functions and structs
- hash_table.c: Implementation of hash table functions
- Makefile: Build configuration
